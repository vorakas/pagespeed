# E2E Kiosk Testing with Emulator

**Source:** LAMPSPLUS Risks section (Asana)
**Status:** Open

## Description
Risk around kiosk E2E testing accuracy when using an emulator rather than physical kiosk hardware. Testing accuracy may be compromised.

## Mitigation
Specific testing plan to be created incorporating testing on physical kiosk devices.
